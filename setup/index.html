Installation

For the purposes of simplicity on this site, we'll deal exclusively with using the GNU open source implementation of the OpenPGP protocol, GnuPG.

Mac

Macs come with a default command-line GPG installation, but you'll want to upgrade to GPG2 for the latest bells and whistles. The easiest way to do so is to grab a copy of Homebrew and run the following: 

$ brew install gpg2

If you prefer working with graphical interfaces you can install GPG Tools.
